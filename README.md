# вҡҷпёҸ Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/CodeAlpha_LanguageTranslationTool.git
```

## Navigate to Project Folder

```bash
cd CodeAlpha_LanguageTranslationTool
```

```bash

```

